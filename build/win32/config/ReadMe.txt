//Configuration file version description.

The execution of the program must depend on the configuration file

//mainnet config
rexx-mainnet.json

//testnet config
rexx-testnet.json

//single config
rexx-single.json


Please copy one of these files to rexx.json first! if not, it can not run

like this:
	cp rexx-mainnet.json rexx.json
or
	cp rexx-testnet.json rexx.json
or
	cp rexx-single.json rexx.json
